การกรองและแปลงข้อมูล (Filter + Map)
- ใช้ filter เลือกเฉพาะข้อมูลที่ตรงตามเงื่อนไข เช่น นักเรียนที่ผ่านคะแนน, นักเรียนที่สอบตก
- ใช้ map แปลงข้อมูล เช่น เปลี่ยนชื่อเป็นตัวพิมพ์ใหญ่/เล็ก หรือรวมชื่อกับอีเมลเป็น string
การเรียงลำดับ (Sort)
- เรียงตัวเลขจากน้อยไปมาก หรือมากไปน้อย
- เรียงชื่อในออบเจกต์ตามตัวอักษรด้วย localeCompare
- การค้นหาและตรวจสอบค่า (Find / FindIndex / Some / Every / IndexOf)
ใช้ find หรือ findIndex เพื่อหาค่าหรือดัชนีแรกที่ตรงเงื่อนไข
ใช้ some เพื่อตรวจสอบว่ามีค่าตรงเงื่อนไขบ้างหรือไม่
ใช้ every เพื่อตรวจสอบว่าทุกค่าตรงเงื่อนไขหรือไม่
ใช้ indexOf หาตำแหน่งของค่าที่ต้องการ
การทำงานกับ Object และ Array
จัดการข้อมูลในรูปแบบออบเจกต์ เช่น ชื่อนักเรียน + คะแนน + อีเมล
ใช้ map และ filter ร่วมกับออบเจกต์เพื่อสร้างผลลัพธ์ใหม่

export = ส่งออกจากไฟล์
import = นำเข้าจากไฟล์
named export = ส่งหลายตัว, ใช้ { }
default export = ส่งตัวหลักตัวเดียว, import ได้ตรง ๆ
